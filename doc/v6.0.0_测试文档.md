# AI养车门店管理系统 v6.0.0 真机测试文档

## 前置条件

### 1. 部署顺序（严格遵守！）

```
第一步：上传并部署云函数 repair_main（必须先完成！）
第二步：确认云函数部署成功（IDE 控制台无报错）
第三步：编译运行小程序进行测试
```

> **⚠️ 如果先发布小程序而云函数未部署，所有 37 个 action 调用会报错！**

### 2. 测试账号准备

| 角色 | 手机号 | 门店码 | 说明 |
|------|--------|--------|------|
| 游客 | 自动进入 | - | 首次打开/清除缓存后自动进入 |
| 免费版管理员 | （你的测试手机号） | （你的门店码） | 未激活 Pro |
| Pro版管理员 | 同上 | 同上 | 已激活 Pro |
| 超级管理员 | 同上 | 同上 | 注册者本人，有 openid |
| 员工(staff) | 需提前创建 | 同上 | 通过 addStaff 创建，role='staff' |

### 3. 测试数据要求

- 门店下至少有 **5 辆车**（含会员和非会员）
- 至少有 **10 条工单**（含施工中/待结算/已完成各种状态）
- 至少有 **3 个会员**
- 至少有 **2 条查车单**
- 至少有 **1 条月报数据**（Pro版测试用）
